Implemented with Minimax alpha-beta pruning

## How to

```bash
# Build
make build

# Run directly with default options
make run
```

## Usage

Run with default depth (6):

```bash
./bin/game
```

Run with custom depth:

```bash
./bin/game 8
```

Valid depth range: 1-12
